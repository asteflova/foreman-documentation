= Configuring {SmartProxyServers} for load balancing

Depending on your {ProjectServer} configuration, you can configure {SmartProxyServers} for load balancing with default or custom SSL certificates and with or without Puppet.

[NOTE]
====
This solution does not use Pacemaker or other similar HA tools to maintain one state across all {SmartProxies}.
To troubleshoot issues, reproduce the issue on each {SmartProxy}, bypassing the load balancer.
====
